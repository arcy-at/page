Images as Block and Inline Elements

What is the difference between a block and inline elements?

Most HTML elements are block elements. For example, the paragraph tag is a block element. They start on a new line and span the width of the document. Heading tags are also block elements. Good examples of inline elements are the italics and emphasis tags and the bold or strong tags. These elements do not start on a new line and only take up as much width as necessary.
